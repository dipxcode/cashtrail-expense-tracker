# CashTrail — Personal Expense Tracker

A full-featured expense and income tracking web app with AI-powered insights, budget management, savings goals, and analytics.

**Live:** https://cashtrail.onrender.com

## Tech Stack

Python · Django 5 · PostgreSQL · Bootstrap 5 · Crispy Forms · Anthropic API · Pandas · Render

## Features

- Track expenses and income with categories, tags, and payment methods
- Budget limits per category with real-time usage tracking
- Savings goals with progress tracking
- AI-powered financial insights (Groq/Gemini API)
- Analytics dashboard with charts
- Admin panel for managing all users and transactions
